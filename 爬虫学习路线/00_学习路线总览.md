# 爬虫技术体系 - 总体学习路线

> 本路线面向有编程基础（Java后端/Vue前端/Python基础）的学习者，采用"原理驱动+项目实战"双轨并行模式。

---

## 学习路径总览

```
第一阶段：核心基础（Chapter 1-4）
├── Chapter 1: Python核心基础（针对爬虫的Python深度用法）
├── Chapter 2: HTTP协议深度剖析（不只是GET/POST，是完整通信链路）
├── Chapter 3: HTML解析与数据提取（选择器/XPath/正则表达式/JSON）
└── Chapter 4: 反爬对抗核心（请求头伪造/IP代理/验证码/Cookie池/频率控制）

第二阶段：JS逆向高阶（Chapter 5-6）
├── Chapter 5: JavaScript核心（针对爬虫的JS语法和浏览器调试）
├── Chapter 6: JS逆向高阶（加密算法还原/AST反混淆/浏览器自动化）

第三阶段：工程化与进阶（Chapter 7-9）
├── Chapter 7: 异步高并发爬虫（aiohttp/协程/百万级数据处理）
├── Chapter 8: 分布式爬虫架构（Scrapy-Redis/消息队列/集群部署）
└── Chapter 9: 项目实战（小说/社交媒体/新闻资讯/政府公开数据）

第四阶段：工程化（Chapter 10）
└── Chapter 10: 合规法则与工程化实践（robots协议/数据存储架构/项目部署）
```

## 各章节学习目标与详细知识点

### Chapter 1: Python核心基础
**学习目标**：掌握爬虫必备的Python高级特性

**详细知识点**：
| 知识点 | 说明 |
|--------|------|
| 协程与异步IO | asyncio/aiohttp实现高并发请求 |
| 连接池与HTTPAdapter | requests的连接复用与性能优化 |
| 上下文管理器 | with语句的资源管理 |
| 生成器与迭代器 | 大数据流式处理 |
| 加密库 | hashlib/hmac/rsa在爬虫中的应用 |
| 实战项目 | 异步并发爬虫框架 |

---

### Chapter 2: HTTP协议深度剖析
**学习目标**：深入理解HTTP通信链路，不只是发请求那么简单

**详细知识点**：
| 知识点 | 说明 |
|--------|------|
| TCP三次握手/四次挥手 | 连接建立与断开过程 |
| HTTPS加密原理 | TLS/SSL握手过程 |
| Cookie/Session/JWT | 三种认证机制的区别与应用 |
| 代理类型 | HTTP/HTTPS/SOCKS5代理 |
| WebSocket | 长连接与实时通信 |
| 连接池原理 | 复用TCP连接的性能优化 |
| 实战项目 | 多平台新闻聚合器（Cookie+JWT认证） |

---

| 正则表达式 | 文本提取与数据清洗 |
| JSON数据处理 | JSON解析与JSONPath |
| **浏览器抓包分析** | **Chrome DevTools/Fiddler定位接口** |
| **接口分析与调用** | **Ajax请求/Fetch请求/接口参数提取** |
| 静态vs动态页面 | 服务器端渲染vs客户端渲染区别 |

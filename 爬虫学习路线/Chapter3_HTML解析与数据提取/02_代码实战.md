## 01_css_selector.py - CSS选择器基础

### 教学目标
掌握BeautifulSoup的CSS选择器语法，能够快速定位HTML元素。

### 核心知识点

```
基础选择器：标签名、.class类名、#id
组合选择器：空格后代、>子元素、+相邻
属性选择器：[attr]、[attr=value]、[attr*=value]
伪类选择器：:first-child、:nth-child(n)
```

### 关键代码解析

**1. 基础三剑客**

```python
from bs4 import BeautifulSoup
soup = BeautifulSoup(html, 'html.parser')

soup.select('div')           # 标签选择器：所有div
soup.select('.news-item')    # 类选择器：class="news-item"
soup.select_one('#header')   # ID选择器：id="header"
```

**2. 组合选择器**

```python
soup.select('.news-list .item')      # 后代：news-list里的item
soup.select('.news-list > .item')    # 子：直接子元素
soup.select('.title + .date')        # 相邻兄弟：title后面的date
```

**3. 属性选择器**

```python
soup.select('a[href]')              # 有href属性的a
soup.select('a[href="/news"]')     # href等于/news
soup.select('[data-id*="2024"]')   # data-id包含2024
```

### 运行方式

```bash
python code/01_css_selector.py
```

## 02_xpath_demo.py - XPath表达式

### 教学目标
掌握lxml库的XPath表达式，能够处理复杂路径和文本匹配。

### 核心知识点

```
路径语法：/绝对路径、//相对路径
谓词条件：[@attr=value]、[position()>n]、[last()]
轴表达式：parent::、following-sibling::、ancestor::
文本函数：text()、contains()、starts-with()
```

### 关键代码解析

**1. 基础路径**

```python
from lxml import etree
tree = etree.HTML(html)

tree.xpath('//div[@class="news"]//a')  # 相对路径查找
tree.xpath('/html/body/div')           # 绝对路径
```

**2. 条件谓词**

```python
tree.xpath('//article[@data-id="1001"]')      # 属性筛选
tree.xpath('//li[position() <= 3]')            # 位置筛选
tree.xpath('//span[contains(@class, "date")]') # 包含筛选
```

**3. 父元素和兄弟**

```python
tree.xpath('//h2/parent::div')                    # 父元素
tree.xpath('//h2/following-sibling::span')        # 后续兄弟
```

### 运行方式

```bash
python code/02_xpath_demo.py
```

## 03_regex_demo.py - 正则表达式

### 教学目标
掌握re模块的正则表达式语法，能够高效提取和清洗文本数据。

### 核心知识点

```
元字符：.任意、\d数字、\w字母数字、\s空白
数量词：*0次+、+1次+、?0或1、{n,m}范围
字符类：[abc]包含、[^abc]不包含
分组捕获：(expr)、(?P<name>expr)
```

### 关键代码解析

**1. 爬虫常用正则**

```python
import re

re.findall(r'https?://\S+', text)          # 提取URL
re.findall(r'\d{4}-\d{2}-\d{2}', text)     # 提取日期
re.findall(r'[a-zA-Z0-9._%+-]+@[\w.-]+', text)  # 提取邮箱
```

**2. 贪婪 vs 非贪婪**

```python
re.search(r'<div>.*</div>', html)    # 贪婪：匹配最长
re.search(r'<div>.*?</div>', html)    # 非贪婪：匹配最短
```

**3. 分组捕获**

```python
match = re.search(r'(\d{4})-(\d{2})-(\d{2})', '2024-01-15')
match.group(1)  # 2024
match.group(2)  # 01
match.group(3)  # 15
```

### 运行方式

```bash
python code/03_regex_demo.py
```

## 04_json_demo.py - JSON数据提取

### 教学目标
掌握JSON数据解析和JSONPath表达式，能够处理API返回的JSON数据。

### 核心知识点

```
json.loads()    字符串转字典
json.dumps()    字典转字符串
JSONPath $       根节点
JSONPath .key    子节点
JSONPath [*]    所有元素
JSONPath [?( @.key)]  条件过滤
```

### 关键代码解析

**1. 安全访问嵌套JSON**

```python
from jsonpath import jsonpath

result = jsonpath(data, '$.data.users[0].name')

# 安全方式（避免KeyError）
name = data.get('user', {}).get('profile', {}).get('name', '默认值')
```

**2. JSONPath条件过滤**

```python
jsonpath(data, '$.data.users[?(@.age>=25)]')  # 年龄>=25的用户
jsonpath(data, '$..name')                       # 递归查找所有name
```

**2. Python复现AJAX请求**

```python
    'Accept': 'application/json',
    'Referer': 'https://target.com/page',
    'User-Agent': 'Mozilla/5.0...'
}

response = requests.get(
    'https://api.target.com/data',
    params={'page': 1, 'limit': 20},
    headers=headers
)

data = response.json()
```

## 06_news_crawler.py - 综合案例：新闻网站

### 教学目标
综合运用CSS选择器、正则表达式、JSON解析，完成一个完整的数据爬取项目。

### 核心知识点

```
requests获取页面
BeautifulSoup解析HTML
CSS选择器提取元素
正则表达式清洗数据
JSON文件存储
```

### 关键代码解析

**1. 完整爬虫流程**

```python
class NewsCrawler:
    def fetch_page(self, url):
        """获取页面"""
        response = requests.get(url, headers=self.headers, timeout=10)
        return response.text

    def parse_news_list(self, html):
        """解析新闻列表"""
        soup = BeautifulSoup(html, 'html.parser')
        news_list = []

        for article in soup.select('.news-item'):
            news = {
                'title': article.select_one('.title a').text.strip(),
                'link': article.select_one('.title a')['href'],
                'date': article.select_one('.date').text.strip(),
                'author': article.select_one('.author').text.strip(),
            }
            news_list.append(news)

        return news_list

    def save_to_json(self, data, filename):
        """保存数据"""
        with open(filename, 'w', encoding='utf-8') as f:
            json.dump(data, f, ensure_ascii=False, indent=2)
```

### 运行方式

```bash
python code/06_news_crawler.py
```

## 项目A：HTTP请求头构造器

### 需求描述

创建一个HTTP请求头构造器，能够分析任意URL并生成完整的浏览器请求头。

### 功能要求

```
1. 输入一个URL，自动分析该网站的请求头需求
2. 生成完整的浏览器请求头（至少10个字段）
3. 能够实际发送请求并验证请求头是否生效
4. 显示响应的状态码、headers和内容
```

### 验收标准

| 标准 | 要求 | 检查方法 |
|------|------|----------|
| 请求头完整 | 包含至少10个常见请求头 | 打印输出查看 |
| User-Agent | 使用真实的浏览器UA | 对比浏览器 |
| Referer | 正确设置防盗链来源 | httpbin.org/headers验证 |
| Accept-Encoding | 支持gzip压缩 | 请求成功且有压缩 |
| Cookie处理 | 能保存和发送Cookie | 两次请求验证 |

### 参考代码结构

```python
"""
HTTP请求头构造器 - 项目A

核心功能：
1. 构造完整的浏览器请求头
2. 发送请求并验证
3. 显示协议分析结果
"""

import requests
import json
from urllib.parse import urlparse

class HeaderBuilder:
    def __init__(self):
        self.default_headers = {
            'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36',
            'Accept': 'text/html,application/xhtml+xml,application/xml;q=0.9,*/*;q=0.8',
            'Accept-Language': 'zh-CN,zh;q=0.9,en;q=0.8',
            'Accept-Encoding': 'gzip, deflate, br',
            'Connection': 'keep-alive',
            'Upgrade-Insecure-Requests': '1',
        }

    def build_headers(self, url, referer=None, cookies=None):
        """构造完整请求头"""
        headers = self.default_headers.copy()

        parsed = urlparse(url)
        headers['Host'] = parsed.netloc

        if referer:
            headers['Referer'] = referer
        else:
            headers['Referer'] = f'{parsed.scheme}://{parsed.netloc}/'

        if cookies:
            headers['Cookie'] = cookies

        return headers

    def analyze_response(self, response):
        """分析响应"""
        print(f"状态码: {response.status_code}")
        print(f"响应头: {json.dumps(dict(response.headers), indent=2, ensure_ascii=False)}")

    def run(self, url):
        headers = self.build_headers(url)
        response = requests.get(url, headers=headers, timeout=10)
        self.analyze_response(response)
        return response


if __name__ == "__main__":
    builder = HeaderBuilder()
    url = input("请输入URL: ")
    builder.run(url)
```

## 项目B：Cookie登录态管理器

### 需求描述

模拟登录流程，管理Cookie和Session，实现登录态的持久化和自动续期。

### 功能要求

```
1. 模拟登录获取Cookie/Session
2. 使用Session自动管理Cookie
3. Cookie持久化到文件
4. 自动检测登录过期并重新登录
5. 多账号切换
```

### 验收标准

| 标准 | 要求 | 检查方法 |
|------|------|----------|
| Session管理 | 后续请求自动带Cookie | httpbin.org/cookies验证 |
| Cookie持久化 | 重启程序后Cookie仍有效 | 保存到文件后重启测试 |
| 登录过期检测 | 401时自动重新登录 | 模拟过期后测试 |
| 多账号支持 | 切换账号互不影响 | 多个Session独立 |

### 参考代码结构

```python
"""
Cookie登录态管理器 - 项目B

核心功能：
1. 模拟登录获取认证Cookie
2. Session自动管理Cookie
3. Cookie持久化存储
4. 自动续期机制
"""

import requests
import json
import os
import time

class LoginManager:
    def __init__(self, account_file="accounts.json"):
        self.session = requests.Session()
        self.account_file = account_file
        self.current_user = None
        self.accounts = self.load_accounts()

    def load_accounts(self):
        """加载账号列表"""
        if os.path.exists(self.account_file):
            with open(self.account_file, 'r', encoding='utf-8') as f:
                return json.load(f)
        return {}

    def save_cookies(self, username):
        """保存Cookie到文件"""
        cookie_file = f"cookies_{username}.json"
        requests.utils.cookiejar_from_dict(
            dict(self.session.cookies),
            cookiejar=None,
            overwrite=True
        )
        self.session.cookies.save(cookie_file)

    def load_cookies(self, username):
        """从文件加载Cookie"""
        cookie_file = f"cookies_{username}.json"
        if os.path.exists(cookie_file):
            self.session.cookies.load(cookie_file)
            return True
        return False

    def login(self, username, password):
        """模拟登录"""
        print(f"正在登录用户: {username}")

        # 模拟登录请求（实际项目中替换为真实网站）
        login_url = "https://httpbin.org/cookies/set/token/abc123 session/user001"

        # 实际登录
        response = self.session.get(login_url)

        if response.status_code == 200:
            self.current_user = username
            self.save_cookies(username)
            print(f"✅ 登录成功")
            return True
        return False

    def is_logged_in(self):
        """检查是否已登录"""
        response = self.session.get("https://httpbin.org/cookies", timeout=5)
        cookies = response.json().get('cookies', {})
        return len(cookies) > 0

    def logout(self):
        """登出"""
        self.session.cookies.clear()
        if self.current_user:
            self.save_cookies(self.current_user)
        self.current_user = None

    def switch_account(self, username):
        """切换账号"""
        self.session = requests.Session()
        if username in self.accounts:
            if self.load_cookies(username):
                if self.is_logged_in():
                    self.current_user = username
                    print(f"✅ 切换到账号: {username}")
                    return True

        print(f"账号 {username} 未登录，正在登录...")
        return self.login(username, self.accounts[username]['password'])

    def run(self):
        """运行演示"""
        print("=" * 50)
        print("Cookie登录态管理器演示")
        print("=" * 50)

        # 登录
        self.login("user1", "pass123")

        # 验证登录状态
        print("\n验证登录状态...")
        response = self.session.get("https://httpbin.org/cookies")
        print(f"当前Cookie: {response.json()}")

        # 保存并重新加载
        print("\n持久化测试...")
        self.save_cookies(self.current_user)

        new_session = requests.Session()
        new_session.cookies.load(f"cookies_{self.current_user}.json")
        response = new_session.get("https://httpbin.org/cookies")
        print(f"重新加载后的Cookie: {response.json()}")


if __name__ == "__main__":
    manager = LoginManager()
    manager.run()
```

## 项目C：代理池与认证管理器

### 需求描述

实现一个支持代理和多种认证方式的爬虫框架，包括HTTP代理、HTTPS代理、Basic认证、Bearer Token和JWT。

### 功能要求

```
1. 支持HTTP/HTTPS代理配置
2. 支持代理认证（用户名密码）
3. 支持Bearer Token认证
4. 支持JWT解码和自动刷新
5. 支持代理轮换和失败重试
```

### 验收标准

| 标准 | 要求 | 检查方法 |
|------|------|----------|
| 代理配置 | 支持HTTP/HTTPS代理 | 实际请求验证IP变化 |
| 代理认证 | 带用户名密码的代理 | httpbin.org/ip验证 |
| Token认证 | Bearer Token自动附加 | httpbin.org/headers验证 |
| JWT解析 | 解码并提取用户信息 | 实际JWT测试 |
| 代理轮换 | 多个代理自动切换 | 日志查看 |

### 参考代码结构

```python
"""
代理池与认证管理器 - 项目C

核心功能：
1. 代理配置和轮换
2. 多种认证方式支持
3. JWT解码和验证
4. 请求重试机制
"""

import requests
import json
import base64
import time
from urllib.parse import urlparse

class ProxyManager:
    def __init__(self):
        self.proxies = []
        self.current_proxy_index = 0
        self.failed_proxies = set()

    def add_proxy(self, proxy_url):
        """添加代理"""
        self.proxies.append(proxy_url)

    def get_next_proxy(self):
        """获取下一个可用代理"""
        available = [p for p in self.proxies if p not in self.failed_proxies]
        if not available:
            self.failed_proxies.clear()
            available = self.proxies

        proxy = available[self.current_proxy_index % len(available)]
        self.current_proxy_index += 1
        return proxy

    def mark_proxy_failed(self, proxy):
        """标记代理失败"""
        self.failed_proxies.add(proxy)


class AuthManager:
    @staticmethod
    def create_basic_auth(username, password):
        """创建Basic认证头"""
        credentials = f"{username}:{password}"
        encoded = base64.b64encode(credentials.encode()).decode()
        return {'Authorization': f'Basic {encoded}'}

    @staticmethod
    def create_bearer_token(token):
        """创建Bearer Token认证头"""
        return {'Authorization': f'Bearer {token}'}

    @staticmethod
    def decode_jwt(token):
        """解码JWT"""
        parts = token.split('.')
        if len(parts) != 3:
            return None

        try:
            header = json.loads(base64.b64decode(parts[0] + '=='))
            payload = json.loads(base64.b64decode(parts[1] + '=='))
            return {'header': header, 'payload': payload}
        except:
            return None

    @staticmethod
    def is_token_expired(payload):
        """检查Token是否过期"""
        if 'exp' not in payload:
            return False
        return payload['exp'] < time.time()


class CrawlerWithProxy:
    def __init__(self):
        self.proxy_manager = ProxyManager()
        self.auth_manager = AuthManager()
        self.session = requests.Session()

    def add_proxy(self, proxy_url, username=None, password=None):
        """添加代理"""
        if username and password:
            proxy_url = f"http://{username}:{password}@{proxy_url.split('://')[-1]}"
        self.proxy_manager.add_proxy(proxy_url)

    def request_with_proxy(self, url, method='GET', auth_type=None, token=None, **kwargs):
        """使用代理发送请求"""
        proxy = self.proxy_manager.get_next_proxy()
        proxies = {'http': proxy, 'https': proxy}

        headers = kwargs.get('headers', {})

        if auth_type == 'basic':
            headers.update(self.auth_manager.create_basic_auth('user', 'pass'))
        elif auth_type == 'bearer' and token:
            headers.update(self.auth_manager.create_bearer_token(token))

        kwargs['headers'] = headers
        kwargs['proxies'] = proxies
        kwargs['timeout'] = kwargs.get('timeout', 10)

        try:
            if method == 'GET':
                response = self.session.get(url, **kwargs)
            else:
                response = self.session.post(url, **kwargs)
            return response
        except Exception as e:
            self.proxy_manager.mark_proxy_failed(proxy)
            raise e

    def run(self):
        """运行演示"""
        print("=" * 50)
        print("代理池与认证管理器演示")
        print("=" * 50)

        # 添加测试代理
        self.add_proxy("http://127.0.0.1:7890")

        # Bearer Token演示
        print("\nBearer Token认证:")
        headers = self.auth_manager.create_bearer_token("test_token_123")
        print(f"  {headers}")

        # JWT解码演示
        print("\nJWT解码:")
        jwt = "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxMjM0NSwidXNlcm5hbWUiOiJ0ZXN0IiwiZXhwIjoxNzA3MDAwMDAwfQ.signature"
        decoded = self.auth_manager.decode_jwt(jwt)
        print(f"  Header: {decoded['header']}")
        print(f"  Payload: {decoded['payload']}")


if __name__ == "__main__":
    crawler = CrawlerWithProxy()
    crawler.run()
```
